# group-sequential-enrichment-two-biomarkers

This repository contains the code to implement the methods in the paper "Bayesian Group Sequential Enrichment Design for Two Biomarkers with Non-Monotonic Treatment Effect".

There are three folders in the repository: "One binary one continuous", "One continuous" and "Two continuous". 

Within each folder, there are two code scripts: Design functions.R and Simulation.R.

Design functions.R includes helper functions to implement the methods.

Simulation.R includes code to run simulations.
